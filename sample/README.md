# lucy [![Build Status](https://secure.travis-ci.org/pentateu/lucy.png?branch=master)](http://travis-ci.org/pentateu/lucy)

## Getting Started
Install the module with: `npm install lucy`

```javascript
var lucy = require('lucy');
lucy.awesome(); // "awesome"
```
